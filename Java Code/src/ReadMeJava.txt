The files contained here are a small part of my intro to database codebase.
Database connections is where all prepared statements are implemented and executed.
Equipment, Members, and Warehouse are all representations of data in the database.
MainMenu, ReportMenu, EquipmentMenu, and RecordMenu are all different menus that the users can interact with to search, insert, and delete data from the database. 
They also display different query results for different use cases.
